🚀 Overview
This project showcases an end-to-end Supply Chain Management Dashboard built in Power BI, designed to deliver holistic visibility across key operational areas and enable data-driven decision-making.
A standout feature of this solution is a custom HTML-based landing page with video integration, providing an engaging and intuitive entry point before navigating into the analytical reports—demonstrating how UI/UX design enhances reporting impact.

🎯 Key Features

📊 Interactive and dynamic dashboards
🎥 Custom HTML landing page with video integration
📌 Advanced KPI tracking using optimized DAX measures
🔍 Drill-down and filter capabilities for deep analysis
🎨 Enhanced UI/UX within Power BI using HTML & CSS


🧩 Dashboard Structure
The solution is organized into four analytical views:
🔹 1. Supply & Quality Control

Cost analysis
Supplier performance tracking
Lead time monitoring
Defect and quality analysis


🔹 2. Manufacturing & Quality Control

Production efficiency metrics
Capacity utilization insights
Throughput analysis
Quality performance tracking


🔹 3. Inventory & Logistics Management

OTIF (On-Time In-Full) performance
Carrier performance analysis
Transportation cost insights
Delivery trends and optimization


🔹 4. Sales & Customer Insights

Revenue and profitability analysis
Product performance tracking
Customer segmentation
Sales trend analysis


🛠️ Tools & Technologies

Power BI Desktop
DAX (Data Analysis Expressions)
Data Modeling & Relationships
HTML/CSS integration within Power BI
Interactive Visualizations & UX Enhancements
